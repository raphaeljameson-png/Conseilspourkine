FICHE KINÉSITHÉRAPIE DÉTAILLÉE — Hernie discale lombaire opérée

Protocole post-discectomie lombaire | 15–25 séances
Version technique kinésithérapeutes
Objectifs : Antalgie • Mobilisation précoce • Gainage transverse/multifide • Chaîne postérieure • Proprioception • Reprise fonctionnelle • Éducation posturale • Prévention récidive (90% dans 6 mois si mal rééduqué)

PROTOCOLE PROGRESSIF PAR PHASES

1

PHASE 1 : J+1 à J+15 (Semaines 1–2) | 2×/sem

Objectif : Protection cicatricielle + antalgie + marche + gainage transverse isométrique

🧘‍♂️ GAINAGE TRANSVERSE

  • Décubitus dorsal • Respiration diaphragmatique
  • Contraction transverse (ventre → nombril) 3×10-15s
  • Progression : 3×20s • ajouter appui genou → planche genoux

🍑 PONT FESSIER ISO

  • Dorsal • genoux 90° • pieds largeur bassin
  • Levée bassin 3×10s isométrique (pas d'hyper-extension)
  • Progression : 1 jambe tendue 3×8s/côté

🚶 MARCHE + POSTURES

  • 10→30min fractionnée (anti-rétroversion)
  • Lever chaise : avant→genoux→bassin→dos
  • Semelles amorties 1-2cm obligatoires

⚡ ANTALGIE

  • Massage cicatriciel J+7 (si non douloureux)
  • TENS 20min/j • Chaleur locale 15min
  • Étirement psoas passif : 3×20s (non radiculaire)
⛔ INTERDITS : Flexion lombaire • Rotation tronc • Charges >5kg • Hyperextension
📋 ÉDUC POSTURALE (Phase 1 obligatoire) :
Lever chaise : avant→genoux→bassin→dos | Port charge : genoux fléchis, dos droit | Conduite : coussin lombaire 4 semaines | Chaussures : semelles amorties 1-2cm
2

PHASE 2 : Semaines 3–6 | 2×/sem (8-12 séances)

Objectif : Gainage dynamique + Chaîne postérieure + Équilibre fonctionnel

🏄 PLANCHES PROGRESSIVES

SemaineExerciceRépétitions
3Planche genoux3×20-30s
4Planche latérale (genou appui)3×15s/côté
5-6Dead bug alterné (jambes fléchies)3×12

💪 CHAÎNE POSTÉRIEURE

  • Fessiers : Pont 1 jambe 3×12/côté
  • Ischios : Curl assis 3×12 (machine ou élastique)
  • Lombaires : Extension dos sol 3×15 (sans hyper-ext.)

⚽ FONCTIONNEL + ÉQUILIBRE

  • Marche rapide 45→60min
  • Squat guidé mur 3×12 (<45°)
  • Équilibre unipodal 3×30s (yeux ouverts)

🧘 ÉTIREMENTS

  • Psoas : 3×30s/côté (genou au sol)
  • Carrés lombes : 3×30s/côté
  • Ischios : 3×30s/côté (dos droit)
⛔ INTERDITS : Charges >10kg • Sport impact • Flexions répétées • Valises
3

PHASE 3 : Semaines 6–12 | 1×/sem (6-8 séances)

Objectif : Gainage avancé + Fonctionnel + Proprioception + Reprise sport

🔥 GAINAGE AVANCÉ

  • Planche complète 3×45s→1min
  • Mountain climbers 3×20-30
  • Gainage TRX suspendu 3×30s
  • Planche dynamique (rotation) 3×20

🏋️ FONCTIONNEL LOURD

  • Squat complet <90° 3×15 (haltères 2-5kg)
  • Soulevé terre roumain 3×12 (barre vide→10kg)
  • Pont fessier 1 jambe lesté 3×15/côté

🎯 PROPRIOCEPTION

  • Bosu équilibre + gainage 3×45s
  • Fentes contrôlées 3×12/côté
  • Unipodal yeux fermés 3×20s/côté
  • Plateau instable + médecine ball 3×30s

🏃 REPRISE SPORT

SemaineActivité
6-8Vélo plat • Natation crawl/dos
8-10Marche rapide • Vélo route
10-12Course légère • Golf • Tennis
12+Contact (accord chirurgien)

CRITÈRES DE SORTIE RÉÉDUC (J+12)

☑ Douleur : VAS <3/10 • Radiculaire absente
☑ Gainage : Planche complète 60s sans compensation
☑ Mobilité : Schober +20cm • SLR >70° sans douleur
☑ Fonction : Squat 90° 3×15 • Marche 60min sans douleur
